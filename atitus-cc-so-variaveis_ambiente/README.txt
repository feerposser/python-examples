Forma de usar:

primeiro, instale as dependência do projeto através do comando
pip install -r requirements.txt

Enquanto isso adicione suas chaves de acesso em variáveis de ambiente

Depois abra 2 terminais e execute os dois comandos abaixo, cada um em um terminal diferente:

python chat-enviar.py
python chat-receber.py

Se estiver utilizando Linux, não esqueça de usar 'python3'
